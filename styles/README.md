- 作成されたファイルは`/ec2/document/styles/dev/{id}.css`に保存される。
- IDがそのまま記述順となる。
    - 大きい数値のIDほど、優先順位が高い。
- セレクタ名は`c`をプレフィクスとして、1000を引いたIDを付ける。
    - `id=1001`なら、`c1`がセレクタ名。
